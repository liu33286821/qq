##遇见问题
  > * 页面滚动的时候 全部播放一栏会置顶
  > * 解决： 把这一栏设置为公共组件 music-common-title  然后在滚动的时候判断距离顶部的位置 显示隐藏`
  > * 之后又碰见问题。  滚动之后  音乐列表最后一个li  高度坍塌。`
  > * 解决： 设置一个开关。  如果向下滚动了 就设置为true  否则就为false 然后在music-common-title 根据传递过来的开关判断是否要添加这个class`
